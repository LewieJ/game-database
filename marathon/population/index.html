About Marathon Population Tracker

Marathon Player Count

Track how many people are playing Marathon right now. Our population tracker pulls live data from Steam and Twitch via API, giving you real-time concurrent player counts and viewership numbers. Whether you're checking if the game is active before jumping in or tracking trends over time, this page has you covered.

Frequently Asked Questions

How often is the player count updated?

Steam player count updates every 30 minutes and Twitch viewership updates every 5 minutes, both via our live API.

Where does the Steam player count come from?

We pull Steam concurrent player data from our backend API, which sources it from publicly available Steam statistics. The number shown is the current count of players in-game.

Does the player count include all platforms?

We currently track Steam concurrent players and Twitch viewership. Marathon supports cross-play across all platforms, but console-specific player counts are not yet available via public APIs.
