What is Game Database?

Hey! I'm not sure how you found this place, but welcome!

gdb.gg is simply a sandbox of tools I've created while learning more about development in general. (Before November 2025, I'd basically never touched a line of code — but here we are!)

My longer-term vision for this platform is pretty simple:

• Build a bunch of tools for everyone to use, free of charge
• If tools exist elsewhere but are poorly maintained or needlessly expensive, find a way to fix that
• Maybe create APIs for popular platforms that don't offer official API support
• Slowly build out a more accessible API platform that anyone can use
• Anything else is a bonus!

Note — and this is important: gdb.gg is a side project. I have a real job outside of this little self-education experiment, and that will always take precedence.
