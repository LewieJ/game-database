What is Game Database?

gdb.gg is a comprehensive gaming statistics platform that tracks player data, game metrics, and leaderboards across multiple gaming platforms.

Our Steam Metrics page provides real-time insights into Steam platform activity, including:

  • Total players online across Steam
  • Live in-game player counts
  • Historical player trends with interactive charts
  • All-time peak records
  • Top 100 games ranked by players

Data Sources:

  • Steam Web API for player counts
  • Historical data collected every 5 minutes
  • Peak records from SteamDB/SteamCharts
